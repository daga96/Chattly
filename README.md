#Chattly
This real-time messaging platform allows users to connect, chat, and share experiences seamlessly.

##Getting Started 
Follow these steps to set up and run the Chat Application locally:

Clone the repository: git clone https://github.com/daga96/Chattly.git

Navigate to the project directory: cd chat-application

Install dependencies for the frontend: cd frontend, cd Chatterly && npm install

Install dependencies for the backend: cd server && npm install

If you want to run building of TailwindCSS: npx tailwindcss -i src/input.css -o public/output.css --watch Start the frontend and backend servers: npm run dev

Open your browser and visit http://localhost:5173 to access the application.

Technologies Used Frontend: React Tailwind CSS

Backend: Node.js Koa.js Socket.IO
